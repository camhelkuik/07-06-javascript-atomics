JS Atomic 06A

True

  • 5 == '5' is true
  • 5 == 2 + 3 is true
  • 5 == 5 is true
  • 5 ===5 is true
  • 5 !== 12 is true
  • 5 < 12 is true
  • 5 <= 12 is true

False

  • 5 == 'five' is false
  • 5 == '6' is false
  • 5 === '5' is false
  • true == 'true' is false
  • true === 'true' is false
  • true == false is false
  • 5 == 12 is false
  • 5 > 12 is false
  • 5 >= 12 is false

What do you call true and false values?

Depends on if you are using the == or ===. A true value for === is only if both sides are equal in value and data type. A true value for == are hard to remember because == tries to convert the values if they are different data types.

How is true similar or dissimilar from "true"? Can they be used in the same way?

True is dissimilar from "true" because one is a boolean and the other is a string. They cannot be used in the same way.
